个人博客搭建小记

发表于 2016-05-24   |   分类于 blog

 之前一直有关注池建强的公众号和廖雪峰、阮一峰的博客，觉得有个自己的博客记录点滴成长是十分必要的，于是就有了此篇小记。

  起初是按照阮一峰老师的教程《搭建一个免费的，无限流量的Blog—-github Pages和Jekyll入门》 来实践，github Pages部分还好，但ruby、gem和jekyll的安装确实有些繁琐，网上的教程零乱加之gem源被墙（btw 后来用了ruby-china的源）等原因走了些弯路，直到找到这个系列教程:《一步步在GitHub上创建博客主页》，总算有了个可靠又比较完整的参考。

  就这样折腾了两天，无奈不知是不是中间来回修改导致某些配置有些冲突，Jekyll-Bootstrap本地部署测试是ok的，但提交到远程仓后Github邮件提示：”The page build failed with the following error:Page build failed.”访问http://username.github.io/ 报404，网上这个问题的资料很少，按照能找到的方法试了还是没解决，正在踌躇时看到了用Hexo搭建博客的教程，简单看了下对Heox的主题比较中意，大家的评价也都不错，鉴于之前已经耗费了不少时间和精力所以果断转战。

 Hexo是一个台湾小哥写的博客框架，官网文档说明很详细，另外参考了《利用GitHub-Pages来Hexo一个独立博客》 ，这次就顺利多了，基于node.js的Hexo安装和配置都比较简单，使用了next 这款主题，初步成果可点击访问我的博客 。

 就先这样了，自己的后院有了，可以周末时细细打理一番，现在工作日还是要好好工作啊。
